## Contracts
- CoinFlip.sol
- Dice.sol
- HighLow.sol
- LuckyDraw.sol
- EvenOdd.sol
- Wheel.sol
- DoubleOrNothing.sol

## ⚠️ Randomness & Fairness
All games use simple onchain randomness without oracles.
Results are NOT provably fair.
This project is educational and experimental.

## Contracts Overview

### CoinFlip
- 50/50 game
- 2x payout

### Dice
- Pick 1–6
- 6x payout

### HighLow
- Low (1–49) / High (52–100)
- ~2x payout

### LuckyDraw
- Fixed odds (1/10)
- 10x payout

### EvenOdd
- Even / Odd
- 2x payout

### Wheel
- Simple wheel (1–10)
- Variable payouts

### DoubleOrNothing
- 50/50 double or lose
